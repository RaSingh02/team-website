Overview

We enjoy cooking, but we find most cooking websites and apps tedious to use. RecipeBuddy is an app that allows the user to store recipes. The user can rate their recipes and indicate the number of times they have cooked the recipe. The user can provide a list of ingredients and the application should return a sorted list of recipes that use those ingredients.
